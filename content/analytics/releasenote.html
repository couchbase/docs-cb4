What’s new?

The following list covers new features in Couchbase Analytics Developer Preview 2.

Issue Description Detail
MB-20446 Configurable query parallelism The degree of parallelism can be set for each individual query.
MB-22051 Resource-based cluster load control New admission control capabilities can prevent the cluster from being overloaded.
MB-22659 Request cancellation A new HTTP endpoint for cancelling an ongoing query and the “cancel” button on the Analytics workbench.
MB-22025 Installation scripts for clusters Support deploying, starting, stopping, and erasing a cluster.
MB-22023 Installation scripts for Amazon Web Services Support starting and stopping a Couchbase Analytics cluster on AWS.

Limitations

The following list covers some of the limitations with Couchbase Analytics Developer Preview 2. Please read this carefully before reporting any new issues.

Issue Description
MB-21155 The SELECT DISTINCT operation does not check object deep equality.
MB-21163 The MINUS, INTERSECT, UNION without ALL operations are not supported.
MB-21182 The OFFSET without a LIMIT statement is not supported.
MB-21184 If you use add a GROUP BY expression in the SQL format, the expression for the GROUP BY key in the SELECT clause must be identical to the GROUP BY key expression.
MB-21270 The PREPARE statement is not supported.
MB-21294 Data Manipulation Language (DML) operations such as INSERT, DELETE, UPSERT, UPDATE do not apply to shadow datasets (which mirror data in Couchbase Server) and are therefore not supported.
