You can use the command line tool, cbq, to run queries. cbq is the command line shell for executing queries against the Query service as well as the Analytics service in Couchbase. It is a builtin tool in Couchbase Server.

First, you should run cbq with the “-e” option to specify the host and port for the Analytics service, for example:

$ cbq -e "http://localhost:8095"

If you’re running Analytics on Docker, execute cbq using the following command:

$ docker exec -it analytics cbq

Then, you can enter the statements in cbq, for example:

cbq> create bucket beerBucket with { "name": "beer-sample" };
cbq> create shadow dataset beers on beerBucket;
cbq> connect bucket beerBucket;
cbq> select count(*) from beers;
