Single Machine Setup

Note: The single machine setup scripts for Analytics are volatile in the Developer Preview. They may change significantly in the future.

Navigating to the “cbas” directory (resulting from unzipping the distribution package), you can start and stop a sample cluster on your local machine by running the following commands.

On Linux and Mac OS X systems:

./opt/local/bin/start-sample-cluster.sh
./opt/local/bin/stop-sample-cluster.sh

On Windows systems:

cd cbas
.\opt\local\bin\start-sample.cluster.bat
.\opt\local\bin\stop-sample-cluster.bat

Then, you should be able to use the Analytics service, through the Analytics workbench, cbq, or the REST API.
